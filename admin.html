Method 2: Auto-Sync via API Key (Advanced)

1. Go to Google Cloud Console and create a project.
2. Enable the Google Drive API.
3. Create an API Key in Credentials.
4. Ensure your Google Drive folder is set to "Anyone with the link can view".
5. Get the Folder ID from your Drive URL (e.g., drive.google.com/drive/folders/YOUR_FOLDER_ID).
